# React Timer App

A simple timer application built using React. This project demonstrates how to use useState and useEffect for handling time-based updates.

---

## Features

- ▶️ Start timer  
- ⏸ Stop timer  
- ⏱ Live time update (HH:MM:SS format)

---

## Concepts Used

- React useState
- React useEffect
- setInterval and cleanup
- Component-based structure

---

## Tech Stack

- React
- Vite

---
